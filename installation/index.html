Installation

Setup

Install the Composer packages:

composer install

Create an .env file:

cp .env.example .env

Update the environment variables in your .env file if necessary:

DB_DRIVER=mysql
DB_HOST=localhost
DB_PORT=3306
DB_DATABASE=chess
DB_USERNAME=root
DB_PASSWORD=

Finally, create the chess database:

php cli/db-create.php
